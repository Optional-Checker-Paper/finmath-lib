13    /**
14    * Interface providing additional methods for
15    * random variable implementing <code>RandomVariable</code>
16    * allowing automatic differentiation.
17    *
18    * The interface will introduce three additional methods:
19    * <code>Long getID()</code> and
20    * <code>Map&lt;Long, RandomVariable&gt; getGradient()</code>
21    * and
22    * <code>Map&lt;Long, RandomVariable&gt; getTangents()</code>.
23    *
24    * The method <code>getGradient</code> will return a map providing the first order
25    * differentiation of the given random variable (this) with respect to
26    * <i>all</i> its input <code>RandomVariableDifferentiable</code>s.
27    *
28    * The method <code>getTangents</code> will return a map providing the first order
29    * differentiation of <i>all</i> dependent random variables with respect to the
30    * given random variable (this).
31    *
32    * To get the differentiation dY/dX of Y with respect to a specific object X using backward mode (getGradient) use
33    * <code>
34    * Map gradient = Y.getGradient();
35    * RandomVariable derivative = Y.get(X.getID());
36    * </code>
37    *
38    * To get the differentiation dY/dX of Y with respect to a specific object X using forward mode (getTanget) use
39    * <code>
40    * Map tangent = X.getTangent();
41    * RandomVariable derivative = X.get(Y.getID());
42    * </code>
43    *
44    * Note: Some implementations may allow limit the result of the gradient to leave nodes or the result of the tangent to terminal nodes.
45    *
46    * @author Christian Fries
47    * @version 1.0
48    */
