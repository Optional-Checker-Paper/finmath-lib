16    /**
17    * This class implements a displaced lognormal model, that is, it provides the drift and volatility specification
18    * and performs the calculation of the numeraire (consistent with the dynamics, i.e. the drift).
19    *
20    * The model is
21    * \[
22    * \mathrm{d}S = r S \mathrm{d}t + \sigma (d \cdot N + S) \mathrm{d}W, \quad S(0) = S_{0},
23    * \]
24    * \[
25    * \mathrm{d}N = r N \mathrm{d}t, \quad N(0) = N_{0},
26    * \]
27    *
28    * Note that
29    * \[
30    * \mathrm{d}(S/N) = \sigma (d+S/N) \mathrm{d}W
31    * \]
32    * that is
33    * \[
34    * \mathrm{d}X = - 1/2 \sigma^2 \mathrm{d}t + \sigma \mathrm{d}W
35    * \]
36    * with exp(X) = d + S/N, i.e. S = N ( exp(X)-d ).
37    *
38    * The class provides the model of S to an <code>{@link net.finmath.montecarlo.process.MonteCarloProcess}</code>
39    * via the specification of
40    * \( S = f(X) = N (exp(X)-d) \), \( \mu = -\frac{1}{2} \sigma^{2} \), \( \lambda_{1,1} = \sigma \), i.e.,
41    * of the SDE
42    * \[
43    * dX = \mu dt + \lambda_{1,1} dW, \quad X(0) = \log(d+S_{0}),
44    * \]
45    * with \( N(0) = 1 \). See {@link net.finmath.montecarlo.process.MonteCarloProcess} for the notation.
46    *
47    * The model can be interpreted as a linear interpolation of the Black-Scholes model
48    * {@link net.finmath.montecarlo.assetderivativevaluation.models.BlackScholesModel}
49    * and the homogeneous Bachelier model
50    * {@link net.finmath.montecarlo.assetderivativevaluation.models.BachelierModel}.
51    *
52    * @author Christian Fries
53    * @see net.finmath.montecarlo.process.MonteCarloProcess The interface for numerical schemes.
54    * @see net.finmath.montecarlo.model.ProcessModel The interface for models provinding parameters to numerical schemes.
55    * @version 1.1
56    */
