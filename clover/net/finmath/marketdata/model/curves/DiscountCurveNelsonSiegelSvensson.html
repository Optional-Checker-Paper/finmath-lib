14    /**
15    * Implementation of a discount factor curve given by a Nelson-Siegel-Svensson (NSS) parameterization.
16    * In the NSS parameterization the zero rate \( r(T) \) is given by
17    *
18    * \[ r(T) = \beta_0 + \beta_1 \frac{1-x_0}{T/\tau_0} + \beta_2 ( \frac{1-x_0}{T/\tau_0} - x_0) + \beta_3 ( \frac{1-x_1}{T/\tau_1} - x_1) \]
19    *
20    * where \( x_0 = \exp(-T/\tau_0) \) and \( x_1 = \exp(-T/\tau_1) \).
21    *
22    * The sub-family of curves with \( \beta_3 = 0 \) is called Nelson-Siegel parameterization.
23    *
24    * Note: This is a time-parameterized model. The finmath lib library uses an internal mapping from date to times \( t \).
25    * This mapping does not necessarily need to correspond with the curves understanding for the parameter \( T \).
26    * For that reason this class allows to re-scale the time parameter. Currently only a simple re-scaling factor is
27    * supported.
28    *
29    * The parameter T used in the parameterization is given by <code>T = timeScaling * t</code>, where t is the maturity as an ACT/365
30    * year fraction from the given reference date.
31    *
32    * @author Christian Fries
33    * @version 1.0
34    */
